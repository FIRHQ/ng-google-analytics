gaType
directive in module fir.analytics

Description

统计指令公用的父类，用于解析name、type、delay、only等所有统计用到的公用参数，推荐加上ga

Usage

as attribute
<ANY ga-type
     ga-type="{string}"
     ga-name="{string}"
     ga-delay="{number|options}"
     ga-only="{boolean|options}">
   ...
</ANY>

Directive info

  • This directive executes at priority level 10.

Parameters

ParamTypeDetails
ga-typestring

对应google analytics Category参数(分类)

ga-namestring

对应google analytics Label参数，用于唯一标识element对象，如果ga-type值为空，将以.截取gaName的值，第一段位type，而后为name

ga-delaynumberoptions

提交延迟，在该值内，发生的连续相同事件将被忽略

ga-onlybooleanoptions

ga-type/ga-name的值是否唯一，默认为true

Methods

  • initGaElement()

    用于获取ga元素对象

Example

<any ga ga-type='example'/>
